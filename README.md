# Loan Express - Lending Management System

A web-based lending management application built with Python and Flask to manage borrower records, daily collections, and profit tracking.

## Features
- Add and manage 400+ borrower records
- Track daily collections and pending interest
- Slot-wise profit rate calculations.
- Real-time balance tracking
- Role-based access (Admin and Viewer)
- Secure login system

## Tech Stack
- **Backend:** Python, Flask
- **Database:** MySQL
- **Frontend:** HTML, CSS, JavaScript

## Note
This project was built for real-world personal use.
